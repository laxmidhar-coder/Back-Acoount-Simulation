# Back-Acoount-Simulation

## Objective
This project simulates basic banking operations using Java Object-Oriented Programming (OOP) concepts such as Classes, Inheritance, Encapsulation, and Method Overriding.

## Features
- Create a bank account
- Deposit money
- Withdraw money
- Check account balance
- Maintain transaction history
- Savings account with minimum balance restriction

## Technologies Used
- Java
- VS Code
- Terminal/Command Prompt

## OOP Concepts Implemented
- Classes and Objects
- Inheritance
- Encapsulation
- Method Overriding
- Constructors
